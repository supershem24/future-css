The :has() CSS Pseudo-class

The CSS pseudo-class :has() makes an element change based on realative selectors. To my knowledge, that means that elements can be affected by what children elements they have or what elements they have directly afterward. This new change allows for a lot of different new possibilities.

For instance:

This is still blue color

   div:has(.frog)
    {
        background-color: #C1FBA4;
    }

This can be used to change outside containers depending on what's inside it, like how in the upper example the frogs caused the div containers to be green. And in the bottom one, the stars affect something new:

They Show
Whether or not
Text is bold 
   div:has(.star)
    {
        font-weight: bold;
    }

If you'd like some more help with this check out :Has() Web Doc for some more info on :has()
